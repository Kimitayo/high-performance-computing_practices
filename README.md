# High Performance Computing Coursework

This repository contains assignments, experiments, notes, and implementations developed during the **High Performance Computing** course offered by the **Instituto Metrópole Digital (IMD)**.

The course focuses on the foundations of **parallel, concurrent, and distributed programming**, as well as **cloud computing**, **supercomputing**, and **hardware architectures for high-performance computing**, especially in the context of **deep learning**.

## Course Information

- **Course Code:** IMD1116
- **Course Name:** High Performance Computing
- **Workload:** 60 hours
- **Responsible Unit:** Instituto Metrópole Digital
- **Course Type:** Module
- **Term:** 2026.1
- **Modality:** In-person

## Course Description

The course covers:

- Fundamentals of parallel programming
- Concurrent and distributed computing
- Cloud computing and supercomputing
- Hardware architectures for high-performance computing
- HPC concepts applied to deep learning workloads

## Objectives

By the end of the course, students are expected to develop a solid understanding of the fundamental concepts of high-performance computing and apply appropriate techniques and methodologies to analyze and solve problems in this area.

The course also aims to encourage:

- Critical thinking
- Autonomy in problem-solving
- Integration of theoretical knowledge with practical challenges
- A broad perspective aligned with industry and academic research demands

## Main Topics Covered

### 1. Performance and Architecture Fundamentals
- Temporal and spatial locality
- Pipeline
- Vectorization
- Memory-bound vs compute-bound workloads

### 2. Parallel Programming
- Parallel program design
- Shared-memory programming
- OpenMP fundamentals
- Task-based parallelism

### 3. Synchronization and Concurrency
- Critical sections
- Named critical regions
- Explicit locks
- Atomic operations
- Producer-consumer problem

### 4. Memory Behavior
- Cache coherence
- False sharing
- Shared memory performance issues

### 5. Numerical and Parallel Computing Applications
- Mathematical approximation of π
- Parallel π computation
- Prime number computation
- Parallel processing of linked lists using OpenMP tasks

### 6. Advanced Context
- Distributed computing concepts
- Cloud computing
- Supercomputing
- HPC hardware for deep learning
